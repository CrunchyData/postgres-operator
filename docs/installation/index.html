Installation

For a quick deployment on either a GKE or OpenShift environment, visit the Quick Installation page.

For a manual installation of the Operator on either a Kubernetes or OpenShift environment, visit the Manual Installation page.

A Helm Chart is also provided.

If you’re looking to upgrade a current PostgreSQL Operator installation, visit the Upgrading the Operator page.

There are many ways to configure the operator further. Some sample configurations are documented on the Configuration page. This includes setting up security and storage configurations for your environment.

After completing the installation steps, ensure you visit the Deployment page to deploy the Operator to your environment.

Next Steps

You may want to find out more information on how the operator is designed to work and deploy. This information can be found in the How It Works page.

Information can be found on the full scope of commands on the Getting Started page.
