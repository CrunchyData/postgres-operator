## pgo create schedule

Create a cron-like scheduled task

### Synopsis

Schedule creates a cron-like scheduled task.  For example:

    pgo create schedule --schedule="* * * * *" --schedule-type=pgbackrest --pgbackrest-backup-type=full mycluster
